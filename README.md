# ACDS - AI Cyber Defense System

This project is a high-fidelity AI-driven Cyber Defense System dashboard and backend.

## Project Structure
- `acds/`: Main project source code.
  - `backend/`: FastAPI backend with log monitoring and threat detection.
  - `frontend/`: React-based frontend dashboard.
  - `frontend_existing/`: Original frontend assets and templates.
- `stitch_screens/`: UI design assets.

## Getting Started

### Backend
1. Navigate to `acds/backend`.
2. Install dependencies: `pip install -r requirements.txt`.
3. Run the server: `python main.py`.

### Frontend
1. Navigate to `acds/frontend`.
2. Install dependencies: `npm install`.
3. Start the dev server: `npm run dev`.

## Features
- Real-time log monitoring and normalization.
- AI-powered threat analysis and playbook generation.
- Interactive threat map and dashboard.
